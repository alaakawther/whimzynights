# Whimzy Nights - Interactive Digital Story

A customized frontend web application that brings a digital story to life through an interactive narrative. The project features a structured sequence of 7 scenes, combining visual assets, text narration, and background audio to create an engaging user experience.

## Key Implementation & Features
- **Dynamic Scene Management:** Built a custom JavaScript workflow to handle seamless transitions between the 7 distinct story acts without page reloads.
- **Audio & Media Integration:** Implemented HTML5 audio elements synchronized with scene changes to deliver immersive background music.
- **Clean UI/UX Layout:** Designed using pure CSS to ensure that navigation controls, story text, and illustrations are aligned and visually appealing.

## 🛠️ Tech Stack & Tools
- **HTML5:** Semantic structure, media tags (audio/image embedding).
- **CSS3:** Custom layouts, transitions, and typography styling.
- **JavaScript (Vanilla):** State handling for scene progression, event listeners for navigation, and audio control.

## 📁 Project Structure Details
The logic is built around managing an array of scenes, where each object contains:
- `narration`: The text content for that specific act.
- `image`: Corresponding visual assets.
- `audio`: Specific sound clips for the atmosphere.
